Flask Weather App
A simple, responsive web application built with Python and Flask that fetches real-time weather data using the OpenWeatherMap API.

Features
Real-time Data: Fetches current temperature, humidity, and weather conditions.

Dynamic Icons: Displays weather-specific icons (sun, clouds, rain, etc.).

Error Handling: Gracefully handles invalid city names or API issues.

Secure: Uses environment variables to protect API credentials.
